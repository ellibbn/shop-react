# pro-react 项目
### 介绍：这是一个基于react实现的商城移动端

## 使用技术介绍：
    1、axios  请求后台数据
    2、fastclick 优化ios无法点击
    3、gulp 优化css（各个浏览器兼容问题）
    4、react-swipe  轮播图
    5、react-dom、react-hot-loader、react-redux、react-router-dom、redux
